Danksagungen

Memory Monitor basiert auf den Quelltexten von LoadInDock, einem CPU-Auslastungsmonitor von Takashi T. Hamada.

Die Anmerkungen über die Mac OS X Speicherverwaltung stammen zum großen Teil aus einem Newsgroup Artikel von Rob Levandowski, erschienen am 28. März 2001 in comp.sys.mac.system.
