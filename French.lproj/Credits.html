Memory Monitor est basé sur le code source de LoadInDock, une application de surveillance d'activité CPU de Takashi T. Hamada.

La note sur la gestion de la mémoire de Mac OS X a été en large part extraite d'un article de newsgroup de Rob Levandowski, posté sur comp.sys.mac.system le 28 mars 2001.
